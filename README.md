# Hand-Written-Digits-Recognition

Identify and predict images of hand written digits using KNN Algorithm also apply PCA to reduce the dimention of the dataset from (42000, 785) to (42000, 2) and (42000, 3) also plot 2D and 3D visulization of the new principle components. The accuracy_score the trained model is about 95%. This is much higher because of sufficiently large value of the size of the dataset
